# AVOCADOS

The main goal of this project

###### 1: Create scroll views inside another scroll view.
###### 2: Use custom View Modifiers.
###### 3: Tab view with custom icons.
###### 4: Master the vertical and horizontal layout design.
###### 5: Support the Light/Dark modes in SwiftUI.
